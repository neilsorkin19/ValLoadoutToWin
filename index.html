Valorant Win Probability Calculator - CMSC320 Final Project

Created by Neil Sorkin - Undergraduate Student at the University of Maryland in CMSC320 (Introduction to Data Science)

Currently, the calculation outputs out a random number, but in the real version of the app, I will use Tensorflow to perform inference using a model I trained with a 9GB dataset of Radiant / Immortal games. During March 2022, I trained this deep learning model to predict the round outcome with 61.7% accuracy before it starts.

In the future, I may add the ability to load preset matches. This way, you don't have to type in each box. I might also add a way to save presets so you can experiment with loadouts.
